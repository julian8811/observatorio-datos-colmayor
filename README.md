# Propuesta Observatorio CTI

Sitio estático generado con `generar_propuesta.py`.

## Cómo abrir

1. **Servidor local (recomendado):** desde esta carpeta ejecuta:
   ```
   python -m http.server 8770 --directory .
   ```
   Luego abre en el navegador: http://localhost:8770/

2. **Archivo directo:** haz doble clic en `index.html` o ábrelo con el navegador (algunos recursos pueden fallar con `file://`).

## Regenerar

```
python generar_propuesta.py
```
